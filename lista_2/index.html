Questão 1

No elemento form, o atributo action especifica a que endpoint enviar os dados fornecidos pelo formulário, enquanto o atributo method especifica o método HTTP a ser usado para enviar os dados: GET ou POST. Fazendo uma analogia, o atributo GET é geralmente utilizado para realizar consultas (e os dados são fornecidos na estrutura da URL), enquanto o atributo POST é geralmente utilizado para realizar inserções (e os dados são fornecidos no corpo da requisição).

Questão 3

O elemento <button> é utilizado para criar um botão simples dentro do documento para realizar uma ação de script, enquanto os elementos <input> são utilizados para criar elementos dentro de um formulário. O elemento <input type="button"> cria um botão similar ao do primeiro caso, utilizado para executar uma ação, enquanto o com <input type="submit"> é utilizado para enviar os dados do formulário, de acordo com o método e ação definidos como atributos dele.
