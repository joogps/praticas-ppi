Questão 2

Input de texto: campo para inserção de texto livre.

Input de número: campo para inserção de números.

Input de email: campo para inserção de endereço de e-mail.

Input de senha: campo para inserção de senha (com bolinhas).

Input de data: campo para inserção de datas.

Input de checkbox: múltipla escolha.

Input "radio": múltipla escolha, mas só pode escolher um.

Textarea: campo de texto com múltiplas linhas.
