Questão 1

Uma regra CSS é um conjunto de instruções que define como os elementos HTML devem ser estilizados no navegador. Ela é composta por um seletor e um bloco de declarações de propriedades. As regras CSS são usadas para controlar a aparência dos elementos na página, como cores, tamanhos de fonte, espaçamento e mais.

Uma propriedade CSS é um atributo que define um aspecto específico do estilo de um elemento HTML. Por exemplo, "color" define a cor do texto, "font-size" define o tamanho da fonte e "margin" define as margens ao redor de um elemento. Qualquer código CSS pode ser atribuído a documentos HTML diretamente em cada elemento, por uma tag style ou por meio de um arquivo externo.

Regras CSS possuem a seguinte forma geral:

seletor { propriedade: valor; }
