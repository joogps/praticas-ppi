As tags para criação de tabelas em HTML são as seguintes: <table>, <tr>, <td> e <th>. A tabela abaixo é um exemplo de tabela em HTML.

      <table>
        <tr>
          <th>Nome</th>
          <th>Idade</th>
        </tr>
        <tr>
          <td>João</td>
          <td>20</td>
        </tr>
        <tr>
          <td>Maria</td>
          <td>30</td>
        </tr>
      </table>
